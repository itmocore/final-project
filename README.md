## Что умеет программа
- добавлять задачи;
- менять и удалять задачи;
- ставить задачи в очередь;
- выполнять задачу из очереди;
- отменять последнее действие;
- выводить задачи по возрастанию дедлайна;
- находить самую раннюю и самую позднюю задачу.

## Команда

**алиса**
- придумала идею и структуру меню;
- сделала ввод данных;

- проверяла, чтобы программа нормально запускалась.

- сделала дерево поиска по дедлайну;

- сделала поиск самой ранней и самой поздней задачи.

## Файлы
- `main.py` — меню и запуск.
- `manager.py` — вся основная логика.
- `task.py` — описание задачи.
- `structures.py` — очередь, стек и дерево.
- `helpers.py` — проверка ввода.
- `README.md` — описание проекта.
- `PROJECT_REPORT.md` — короткий отчёт.

## Запуск
```bash
python main.py
```
